BGM
----------------------------------------------------------------------
0 : No01 -> bgm_No01_TitleAndMenu     (タイトル、メニュー画面)
1 : No03 -> bgm_No02_CreateAndPrint   (クリエイト、プリント画面)
2 : No06 -> bgm_No03_Printing         (プリント中)
3 : No02 -> bgm_No04_MiniGame         (ミニゲーム画面)

----------------------------------------------------------------------
全てループ仕様です
再生時にループ指定するのを忘れないでください

AudioPlayer クラスの Play() メソッドから指定できます

SE
----------------------------------------------------------------------
4 : No04 -> se_No05_CharacterRotation (キャラ回転)
5 : No05 -> se_No06_ChangeDesign      (キャラ見た目変更)

  : No07 -> ※仕様決定後にナンバリング

6 : No08 -> se_No07_Next              (進む：戻る以外のボタン押したとき)
7 : No09 -> se_No08_Back              (戻る)
8 : No10 -> se_No09_OK                (選択：決定するボタン押したとき)

9 : No11 -> se_No10_TitleLogoSetup    (タイトルロゴ組み立て音)
10: No12 -> se_No11_CharacterFallDown (キャラ落下：タイトルロゴ組み立て完了後)
11: No13 -> se_No12_CharacterLanding  (キャラ着地：落下してきたキャラの着地)
12: No14 -> se_No13_ExpandBox         (メニュー画面のボックス展開)

  : No15 -> 仕様変更のため削除（後日追加なら決定後にナンバリング）

13: No33 -> se_No14_Jump              (タイトル、クリエイトのジャンプ音)

14: No17 -> se_No15_StartCountDown    (ゲーム開始時のカウントダウン)
15: No18 -> se_No16_FinishCountDown   (ゲーム終了前のカウントダウン)
16: No19 -> se_No17_Finish            (ゲーム終了のホイッスル)
17: No20 -> se_No18_CostumeBreak_A    (リザルト発表時のコスチュームが吹っ飛ぶ音１)
18: No20 -> se_No19_CostumeBreak_B    (リザルト発表時のコスチュームが吹っ飛ぶ音２)
19: No21 -> se_No20_Result            (リザルトの表示)

20: No22 -> se_No21_Shot              (スピードのゲームでショット発射時)
21: No23 -> se_No22_Hit               (スピートのゲームでショットが当たった時)
